En los siguientes ejercicios debes utilizar escuchadores de eventos para manejarlos.
Por ejemplo:

Obtenemos la referencia a un elemento HTML:

const boton = document.getElementById("boton");

Manejamos el evento "click":

// Con referencia a función
boton.addEventListener("click", funcion_Manejadora_del_Evento );
// Con función flecha
boton.addEventListener("click", () => {
 ... 
});
// Con función "normal"
boton.addEventListener("click", function() {
	...
});

Si necesitas información del evento, debes recogerlo como argumento (ejercicio 21)

boton.addEventListener("click", ev => {
 ...
});
boton.addEventListener("click", function(ev) {
	...
});

  2. chat para solitarios
    Crea un formulario como el de la figura, tendrá un textarea un input-text y un botón.

    Lo que se vaya escribiendo en el cuadro de texto se irá añadiendo al textarea cada vez que pulsemos el botón.

    - Cada vez que se pulse el botón también se vaciará el cuadro de texto, y se le devolverá el enfoque.

    Para recuperar el foco en el cuadro de texto cuando se puls "Enviar", el input text debe invocar su método focus():

    inpTxt.focus();

  3. keydown

    Cada vez que pulsamos una tecla se generan los eventos keydown y keyup.
    (keydown en MDN).

    Modifica el chat de manera que, cada vez que se pulse la tecla Enter en el cuadro de texto se lleven a cabo las mismas acciones que cuando se pulsa el botón "Enviar".

    Abajo tienes un ejemplo de uso de keydown y detección de la tecla Enter:

    // en la propiedad "key" del evento, se guarda la tecla pulsada
inpTxt.addEventListener( "keydown", ev => {
   if(ev.key == "Enter") alert("Has pulsado Enter");
})

  2. crono botón
    Crea una página web con un único botón. La primera vez que se pulse mostrará el texto: Cronómetro iniciado

    A partir de ahora, cada pulsación mostrará el tiempo transcurrido desde que se pulsó por primera vez.

    Los objetos Date almacenan los milisegundos transcurridos desde el 1-ene-1970 00:00:00.
    Puedes obtenerlos con su método static now().
