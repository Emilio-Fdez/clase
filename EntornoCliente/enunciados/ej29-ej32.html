  1. función para visualizar arrays
    Crea una function showArray(Array) que visualice los elementos de un array de 1 ó 2 dimensines.
     

    Ejemplo:

    const uno = [ 1, 2, 2 ];
const dos = [ [1, 2], [1, 2, 3], [1, 2, 3, 4] ];

showArray(uno)
   1 2 2
showArray(dos)
   1 2
   1 2 3
   1 2 3 4
