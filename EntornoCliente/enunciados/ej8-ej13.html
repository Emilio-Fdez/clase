Ejercicios 7-octubre

- En los ejercicios no se realizará código HTML con JavaScript, salvo que se pida expresamente.
El JavaScript se codificará en hoja aparte.

- La etiqueta <script> irá en la cabecera (<head>) del documento (no olvidar usar el atributo defer)

- Todo el código deberá estar bien formado y tabulado.
 

  1. Página de color personalizado.

    Muestra la página web con los colores de texto y fondo que te pida el usuario desde dos input-text y un botón para aplicar.

    Por defecto, en los input se mostrarán los colores black y white.

    Para acceder a las propiedades CSS de un elemento se usa el atributo style, por ejemplo:

    let h1 = document.getElementById("titulo");
h1.style.color = "red";
h1.style.backgroundColor = "#EA6322";

    O bien:

    document.querySelector("p").style = "background-color: yellow; color: orange";

    Cambia uno de los input-text por un input-color y comprueba que funciona exactamente igual.


  2. password - Página web para solicitar un password. Tendrá un input-text y un botón.

    Dicho password se almacenará en una variable, y será el siguiente:

    password: secreto

    Se admitirán hasta tres intentos de acierto.
    - Si errara se dará por finalizado el proceso de login mostrando un mensaje en la página y deshabilitando el botón:.

      document.querySelector("#boton").disabled = true;
    - Si acertara el password, se mostrará un mensaje de bienvenida.

  3. ocultar foto - Crea una página con un botón y una fotografía. Al clickar en la foto se ocultará, y con el botón se volverá a visibilizar.

    ocultar un elemento de la página web
    - estilos CSS: visibility: { hidden | visible }
    display: { none | inline | block | inline-block ...}
    - atributo HTML: hidden="{ true | false }"


  4. <table> de multiplicar.
    Genera con JavaScript el código HTML necesario que muestre la tabla de multiplicar del 1 al 10 (necesitarás dos bucles for, anidados)
    En la primera fila y en la primera columna, se mostrarán los multiplicandos.
    Por ejemplo:

    ×123
    1123
    2246
    3369

    Es posible que el navegador intente renderizar correctamente la tabla si la vas escribiendo poco a poco sobre el documento, obteniéndose un resultado no deseado.

    Guarda todo el HTML que conforma la tabla en una variable String, y luego plásmalo sobre la página web en el innerHTML de algún elemento.

  5. Operador ternario.

    Utilizando un único operador ternario muestra la calificación en función de una nota. Nota menor de 5: Suspenso Nota de 5 a algo menor de 7: Aprobado Nota de 7 a algo menor de 9: Notable Nota entre 9 y 10: Sobresaiente

    Debe validarse que el usuario proporciona una entrada numérica y además está comprendida entre 0 y 10.
    En caso de que la entrada sea incorrecta, se informará al usuario del error cometido.
     

  6. Imágenes en posición aleatoria. Sitúa 3 imágenes en la página web que cambien a una posición aleatoria cada vez que se pulsa un botón.

    Necesitas las propiedades CSS:
    position: absolute top: <distancia al margen superior> left: <distancia al margen izquierdo>

    El ancho/alto de la parte de la ventana que visualiza el documento, se puede obtener con:
    window.innerWidth
    window.innerHeight
